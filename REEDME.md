This project analyzes a public Kaggle sales dataset using SQLite.
The objective is to perform an initial exploratory data analysis (EDA) with SQL and use Tableau to visualize the main business insights.

### 1. Project Overview

* Explore and understand the structure of the dataset

* Generate summary statistics using SQL

* Identify top-performing products, months and customer segments

* Create a Tableau dashboard to present key insights

### 2. SQL Analysis

The SQL analysis includes:

* Table overview

* Total number of rows

* Number of unique products

* Number of unique categories and subcategories

* Number of unique states

Descriptive statistics

* Average order quantity

* Average profit

* Profit insights

* Profit by age group

* Top 10 products by profit

* Monthly profit with correct month order

* Profit by product category

* Profit by country

All SQL queries used in the project can be found in the file:
sql/sales_analysis.sql

### 3. Tableau Dashboard

The dashboard visualizes the most important metrics and insights, including:

* Profit by product category

* Profit by month

* Profit by country

### 4. Tools Used

* SQLite for data exploration and analysis

* GitHub for project documentation

### 5. Summary

This project demonstrates basic data analysis skills, including SQL data exploration, identification of key business insights, creation of meaningful visualizations and clear documentation suitable for recruiters or portfolio usage.
